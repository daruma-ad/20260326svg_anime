# 静止画に命を吹き込む：SVGアニメーションで広がるWebデザインの可能性

Webデザインにおいて、ユーザーの目を引き、体験を豊かにするために「動き」は欠かせない要素です。今回は、無料で高品質なイラストレーションを提供している [unDraw](https://undraw.co/illustrations) のSVG素材をベースに、CSSアニメーションを加えて「命を吹き込む」プロセスをご紹介します。

## 1. なぜSVGアニメーションなのか？

SVG（Scalable Vector Graphics）は、その名の通り拡大しても画質が劣化しないベクター形式の画像です。しかし、最大のメリットは「コードで記述されている」点にあります。

- **軽量:** 動画やGIFアニメーションに比べてファイルサイズが圧倒的に小さい。
- **制御不能な拡張性:** CSSやJavaScriptを使って、画像内の特定のパーツだけを動かすことができる。
- **高解像度:** どんなディスプレイでも常に鮮明。

今回は、この特性を活かして、1枚の静止画イラストを「物語を感じるデザイン」へと進化させました。

## 2. 制作のステップ：星座観測イラストの進化

### ベース素材の選択
[unDraw](https://undraw.co/illustrations) から「Astronomy（天文学）」をテーマにしたイラストをピックアップしました。このままでも十分に素晴らしいデザインですが、夜空の「星」が静止しているのが少しもったいないと感じました。

### CSSによる「瞬き」の実装
SVGのソースコードを解析し、星を表すパーツにクラスを付与。CSSの `@keyframes` を使って、透明度 (`opacity`) を不規則に変化させるアニメーションを適用しました。

```css
@keyframes twinkle {
  0%, 80% { opacity: 0; }
  85% { opacity: 1; }
  90%, 100% { opacity: 0; }
}
```

ポイントは、すべての星を同じタイミングで動かさないことです。`animation-delay` にマイナスの値をランダムに設定することで、ページを開いた瞬間からそれぞれの星が異なるリズムで瞬き始める「自然な夜空」を表現しました。

### 世界観を補完するデザイン
アニメーションしたSVGを、最新のトレンドである「グラスモーフィズム（硝子のような質感）」を取り入れたホームページに組み込みました。背景に揺らめく光のオーブを配置することで、SVG内の星の瞬きがより一層引き立つ構成にしています。

## 3. 結論：小さな動きが大きな印象の違いを生む

今回行った修正は、コード数行の追加に過ぎません。しかし、ただの「画像」が「生きているシーン」に変わることで、サイト全体の高級感とストーリー性が一気に高まりました。

[unDraw](https://undraw.co/illustrations) のような優れた素材サイトと、少しのCSSアニメーションの知識があれば、誰でもプロフェッショナルで魅力的なWebエクスペリエンスを構築できます。皆さんも、手元のSVGに「命」を吹き込んでみませんか？
